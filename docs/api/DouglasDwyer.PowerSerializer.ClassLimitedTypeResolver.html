Class ClassLimitedTypeResolver

Instances of FinalizerLimitedTypeResolver allow for serialization of all the classes specified in the constructor, as well as all structs loaded at resolver construction. They utilize the MD5 hash of a type's name in order to resolve it across serialization boundaries.
