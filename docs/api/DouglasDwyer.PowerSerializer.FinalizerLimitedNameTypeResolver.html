Class FinalizerLimitedNameTypeResolver

Instances of FinalizerLimitedNameTypeResolver allow for serialization of any type loaded in the current System.AppDomain upon resolver construction, provided that the type doesn't have a finalizer. They utilize the a type's assembly-qualified name in order to resolve it across serialization boundaries.
