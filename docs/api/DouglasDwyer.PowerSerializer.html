Namespace DouglasDwyer.PowerSerializer

Classes

ClassLimitedTypeResolver

Instances of FinalizerLimitedGuidTypeResolver allow for serialization of all the classes specified in the constructor, as well as all structs loaded at resolver construction. They utilize the MD5 hash of a type's name in order to resolve it across serialization boundaries.

FinalizerLimitedGuidTypeResolver

Instances of FinalizerLimitedGuidTypeResolver allow for serialization of any type loaded in the current System.AppDomain upon resolver construction, provided that the type doesn't have a finalizer. They utilize the MD5 hash of a type's name in order to resolve it across serialization boundaries.

FinalizerLimitedNameTypeResolver

Instances of FinalizerLimitedNameTypeResolver allow for serialization of any type loaded in the current System.AppDomain upon resolver construction, provided that the type doesn't have a finalizer. They utilize the a type's assembly-qualified name in order to resolve it across serialization boundaries.

FullGuidTypeResolver

Instances of FullGuidTypeResolver allow for serialization of any and all types loaded in the current System.AppDomain upon resolver construction. They utilize the MD5 hash of a type's name in order to resolve it across serialization boundaries.

GuidTypeResolver

Instances of GuidTypeResolver utilize the MD5 hash of a type's name in order to resolve it across serialization boundaries. They allow for serialization of any type specified in the constructor.

NameTypeResolver

Instances of NameTypeResolver utilize the a type's assembly-qualified name in order to resolve it across serialization boundaries. They allow for serialization of all types.

PowerDeserializationContext

Stores data about a PowerSerializer's deserialization operation, including a list of serialized types as well as the current object graph. This class, along with CreateDeserializationContext(), may be overriden to store additional information.

PowerSerializationContext

Stores data about a PowerSerializer's serialization operation, including a list of serialized types as well as the current object graph. This class, along with CreateSerializationContext(), may be overriden to store additional information.

PowerSerializer

Allows for customizable binary serialization and deserialization of complex types. Instances of PowerSerializer can serialize absolutely anything, provided that their type resolver allows it.

Interfaces

ITypeResolver

Types that implement ITypeResolver mediate which types can be serialized/deserialized by a PowerSerializer. Additionally, they each provide a way to identify types across serialization boundaries.
